    Documentation Home

    Welcome to the Phylocanvas documentation site. Please note that this site is intended for web developers and assumes a knowledge of modern web development workflow practises. These pages detail how to use the Phylocanvas version 2 and above, but the v1.x docs are still available.

    Sections

    To get a demo up and running, see Quick Start.

    To start using Phylocanvas in your app, see Installation.

    To customise trees and interactions, see Features.

    To respond to user interactions with a tree, see Events.

    To extend the abilities of Phylocanvas, see Plugins.

    To start using v2.x from a previous version, see Migrating from v1.x.

    To make improvements to Phylocanvas, see Development.
